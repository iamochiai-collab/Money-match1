# お金をあわせよう ランダム商品版

SafariやChromeで `index.html` を開くと使えます。

## 内容
- 「つぎへ」を押すと、品物のイラストと金額がランダムに出ます。
- 500円玉×1、100円玉×4、50円玉×1、10円玉×4、5円玉×1、1円玉×4から、必要な硬貨を動かして支払いを作ります。
- 合計金額の表示はありません。
- 「こたえあわせ」で正誤判定します。
- 硬貨はドラッグ、またはタップで移動できます。

## 金額や品物を変える方法
`index.html` の `QUESTIONS` の中を編集してください。
例：
`{name:'ジュース', amount:120, color:'#ff8a00', svg:'drink'}`
