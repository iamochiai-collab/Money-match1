# お金をあわせよう レベル選択版

SafariやChromeで `index.html` を開くと使えます。GitHub Pagesにもそのまま公開できます。

## 内容
- レベル1・レベル2・レベル3を選べます。
- 各レベルで約200問のランダム問題を自動生成します。
- 「つぎへ」を押すと、品物のイラストと金額がランダムに出ます。
- 合計金額の表示はありません。
- 硬貨はドラッグ、またはタップで「ここにお金をはこぶ」へ移動できます。
- 移動した硬貨は「えらべるお金」の枚数表示から減ります。
- 戻すと枚数表示も戻ります。
- 「こたえあわせ」で正誤判定します。

## レベル
- レベル1：500円玉・50円玉・5円玉は使わず、2種類の硬貨でできる問題。
- レベル2：500円玉以外を使って、複数の硬貨を組み合わせる問題。
- レベル3：111円〜999円まで、500円玉も使うチャレンジ問題。

## ファイル構成
- `index.html`：アプリ本体
- `assets/`：硬貨画像
- `manifest.webmanifest`：ホーム画面追加用
- `sw.js`：簡易キャッシュ用
